# Security Policy / 安全策略

**请不要在公开 Issue / PR / 讨论区披露安全问题。**
**Please do NOT disclose security issues in public Issues / PRs / discussions.**

如果你发现赛题官网或比赛基础设施中的安全问题(例如榜单数据可被篡改、提交鉴权可绕过、
可读取他人 token 等),请通过官网公布的官方联系方式**私下**报告给 TronCamp 主办方,并附
影响范围、复现步骤与影响评估。修复发布前请勿公开细节。

If you find a security issue in the competition website or infrastructure (e.g.
leaderboard tampering, submission-auth bypass, token leakage), please report it
**privately** to the TronCamp organizers via the official contact channel listed on the
site, including scope, reproduction steps, and impact. Please do not disclose publicly
until a fix has been released.
